Sync waves#

MonitoringChart annotates every resource with an argocd.argoproj.io/sync-wave value. ArgoCD applies lower waves before higher waves, which gives the stack a deterministic rollout order. The waves run from 0 to 3.

Application dashboards and alerts#

The library does not place application dashboards or alerts in any wave; it ships none. Integration charts attach their own dashboard and alert constructs and choose a wave for them. Wave 3 is a natural fit, since the dashboards are discovered after Grafana is running. See Add application dashboards and alerts.
